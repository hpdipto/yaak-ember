# Ember Theme for Yaak

A warm, monochrome color theme for [Yaak](https://yaak.app) based on [Ember](https://embertheme.com/).

Includes two variants:

- **Ember Soft** — dark theme with a warm graphite foundation
- **Ember Light** — light theme on warm parchment

## Credits

Color palette and design by [Ember Theme](https://embertheme.com/). This plugin adapts the palette for Yaak.
